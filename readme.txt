The smart plugin to insert custom header and footer scripts to your website.
  
== Description ==
  
ScriptPress allows you to add your custom css and javascript codes to the header and footer sections of your website.

ScriptPress is the ideal plugin to add Google Adsense snippets, analytics scripts, and the likes, to the head of your website.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "ScriptPress"
3. **Install and Activate** ScriptPress from your Plugins page
4. **Click** on the new menu item "ScriptPress" and use the provided tabs to insert your custom scripts to your site.

  
== Frequently Asked Questions ==
  
= How do I use this plugin? =
  
1. Click on the ScriptPress menu at the left admin menu bar.
2. You will find the self-explanatory tabs to insert your scripts on the admin page
  
= How to uninstall the plugin? =
  
Simply deactivate and delete the plugin.
